## Grounding docs (pinned sources)

This directory is reserved for **pinned source-of-truth notes** that power the `docs` MCP server.

Expected contents (to be added as you select vendors and datasets):
- AU HI/IHI integration notes (endpoint choice, auth mode, compliance constraints)
- Terminology source notes (SNOMED CT / ICD-11: hosting, licensing, versions)
- Guideline corpus notes (Choosing Wisely region + licensing; eTG licensed adapter constraints)
- Pharmacy directory source notes (coverage, ToS, caching)

Every note must include:
- **Source** (URL or licensed asset reference)
- **Version/date**
- **Usage constraints** (license terms, access control)
